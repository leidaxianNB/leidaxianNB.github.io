前言

本章开始记录的是学习C#的第一天，一边学习一边更新记录。

介绍

.NET dotnet平台

微软Microsoft 新一代多语言的开发平台，用于构建和应用程序

C sharp

专门为.NET推出的高级语言，2000就开发了，2002发布1.0

Mono

Novell公司支持在其他操作系统下开发的.NET程序框架

Unity借助Mono实现跨平台，核心就是.NET Framework 框架。

定义

程序运行的地方 –内存

程序处理的东西 –数据

内存中开辟的一块用于储存数据的空间 –变量

容量单位：

位bit（比特） – 电脑记忆体最小单位，每一位代表0或1
字节byte – 电脑存储最小的单位

1 Byte = 8 Bit

1KB = 1024 Byte

网速 20M 指的是20 Mbps (兆位/秒)是速率，换算字节应该除8，所以应该是20/8 = 2.5 m/s (兆字节/秒)

数据类型：

整形

int(整数) 4个字节 有符号（-2的32次方2的32次方），无符号为（02的64次方）

非整形

float（小数） 单精度浮点，4字节，精度7位
double 双精度浮点，8字节，精度15-16位
decimal 16字节，精度28-29位

非数值型

char 字符，2字节，存储单个字符，使用单引号。

string 字符串，存储文本，使用双引号

bool 类型，1字节，可以直接赋值true真false假，做判断。

声明

规则：

1.不能以数字开头，只能字母、数字、下划线构成

2.不能用关键字（蓝色的）

3.开头最好是小写字母

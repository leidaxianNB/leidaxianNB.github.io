Stable Diffusion安装

Created2024-05-23|Updated2024-06-18|Stable Diffusion
|Post Views:

前言

Stable Diffusion是基于潜在扩散模型的文本到图像生成模型，可以根据文本描述产生详细图像。人话就是AI绘画。

以前Stable Diffusion开源出来的只是很长很复杂的源代码，但是后面有一位越南大佬（automatic1111）把代码整合成基于网页运行的应用程序：Stable Diffusion Web UI

本篇记录的是如何安装SD的两种：一种是automatic1111的项目自定义安装（最基本）；另一种是整合包（B站UP 秋葉aaaki）

秋叶整合包适合新手，而auto的整合包最简洁，自定义安装。

自定义安装

首先从github下载项目

也可以从文件夹中打开终端，输入：

进行克隆

接着从python官网下载python3.10.6版本(推荐)

安装程序界面要记得点击add Python XXX to PATH (这里是添加环境变量到系统)

还需要下载git

都下载完后打开项目，找到 webui-user.bat，点击后开始下载必要依赖


中途可能要重新打开，等待安装后，直到看到本地部署的网址，将这个网址复制到浏览器打开就会出现stable diffusion web ui了

http://127.0.0.1:7860 就是网址

秋叶整合包

首先下载整合包.(123wp)

模型，依赖安装程序，还有一个整合包。

下载好后点击启动依赖等待安装

接着打开整合包4.8，运行绘世启动器.exe

点击右下角一键启动

点击后为 运行中

等待终端加载启动项

最后加载完成后，就会以网页的形式展现出stable diffusion web ui。

可以在提示词里输入a cat是否成功生成一只猫，这样就完成安装了。

至此两种安装方式完成。后面就可以自己下载你喜欢的模型来生成你喜欢的图啦。

注意

如果是mac端的话，自定义安装会有所不一样，因为我所使用的是window系统不了解mac，所以最好选择整合包。

END
